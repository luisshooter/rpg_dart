# RPG Flutter Application

This project is a Flutter-based RPG (Role-Playing Game) application that provides an interactive and engaging experience for users to create, manage, and battle with RPG characters. The application leverages Flutter's powerful UI capabilities to deliver a smooth and modern experience across multiple platforms such as Android, iOS, and others.

## Features

- Create and customize RPG characters with different races and archetypes.
- Manage a list of characters with editing and deleting functionalities.
- Interactive and user-friendly UI with modern design elements.
- Character attributes allocation with real-time validation.
- Navigation between Arena, Characters, and History screens.

## Installation

1. Ensure you have Flutter SDK installed. For installation instructions, visit [Flutter Install](https://flutter.dev/docs/get-started/install).
2. Clone this repository:
   ```
   git clone <repository-url>
   ```
3. Navigate to the project directory:
   ```
   cd rpg_app-aula-18-11-2025
   ```
4. Get the dependencies:
   ```
   flutter pub get
   ```
5. Run the app:
   ```
   flutter run
   ```

## Usage

- Use the bottom navigation to switch between Arena, Characters, and History.
- Use the floating action button on the Characters screen to add a new hero.
- Input hero details including name, race, archetype, and allocate points to stats.
- Save heroes and manage them through the list interface.
