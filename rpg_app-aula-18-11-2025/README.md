# Aplicativo RPG Flutter

Este projeto é um aplicativo RPG (Role-Playing Game) desenvolvido com Flutter, que oferece uma experiência interativa e envolvente para os usuários criarem, gerenciarem e batalharem com personagens RPG. O aplicativo utiliza os recursos poderosos do Flutter para proporcionar uma experiência moderna e fluida em múltiplas plataformas, como Android, iOS e outras.

## Funcionalidades

- Criação e customização de personagens RPG com diferentes raças e arquétipos.
- Gerenciamento da lista de personagens com funcionalidades de edição e exclusão.
- Interface interativa e amigável com elementos visuais modernos.
- Alocação de atributos dos personagens com validação em tempo real.
- Navegação entre as telas Arena, Personagens e Histórico.

## Instalação

1. Garanta que o SDK do Flutter esteja instalado. Para instruções, visite [Flutter Install](https://flutter.dev/docs/get-started/install).
2. Clone este repositório:
   ```
   git clone <url-do-repositório>
   ```
3. Navegue até o diretório do projeto:
   ```
   cd rpg_app-aula-18-11-2025
   ```
4. Instale as dependências:
   ```
   flutter pub get
   ```
5. Execute o aplicativo:
   ```
   flutter run
   ```

## Uso

- Use a navegação inferior para alternar entre Arena, Personagens e Histórico.
- Utilize o botão flutuante na tela de Personagens para adicionar um novo herói.
- Insira os detalhes do herói, incluindo nome, raça, arquétipo e aloque pontos nos atributos.
- Salve os heróis e gerencie-os através da lista.
